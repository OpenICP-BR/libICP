NAME

       libICP - An ICP-Brasil CAdES library (part of the OpenICP-BR project)

SYNOPSIS

       libICP is a golang library for processing ICP-Brasil CAdES digital
       signatures.

       These man pages document the C wrapper for the original library.

COMPILATION

       To compile a program that uses this library, simply run:

         gcc my_program.c -o my_program /path/to/installation/libICP.a -I/path/to/installation
