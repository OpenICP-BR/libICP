NAME

       icp_errc - A coded error

SYNOPSIS

       A type that describes an error and carries both a text message and an
       error code. This last part is available as an int and as a string.

       All icp_errc "inherit" icp_err(3).

SEE ALSO

       libICP(3) - General library documentation

       icp_errc_code(3) - To get the int code

       icp_errc_code_str(3) - To get the string

       icp_err_str(3) - To get the text message
